Revision Program

Revision program for GCSE and A-Level students. Jan 2024 - May 2024

C#

SQL

Microsoft Access

.NET

I created a Windows Forms app in C# to help GCSE and A-Level students revise more effectively by eliminating distractions. I conducted surveys with students and teachers to identify key features, then developed a backend database using Microsoft Access and SQL to store user accounts and subjects.
To ensure data security, I implemented encryption to hash and salt passwords. The app was tested extensively by users, and I used their feedback to refine and improve the app through an iterative Rapid Application Development process.
